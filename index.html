AdaPCLA Anonymous Release

Anonymous code release for the paper AdaPCLA: Curriculum Prior Internalization for Long-Tailed Longitudinal EHR Generation.

The code package includes training, ablation, analysis, and zero-shot control scripts. Raw MIMIC data and pretrained checkpoints are not included.

pip install -r requirements.txt
